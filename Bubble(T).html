What is Bubble Sort?

Bubble Sort is a simple sorting algorithm that repeatedly steps through the list to be sorted, compares each pair of adjacent items and swaps them if they are in the wrong order. The pass through the list is repeated until the list is sorted.

How does Bubble Sort work?

  1. Step 1: Compare the first two elements in the array.
  2. Step 2: If the first element is larger than the second element, swap them.
  3. Step 3: Move to the next pair and repeat the process until the end of the array is reached.
  4. Step 4: After each pass, the largest element "bubbles up" to its correct position.
  5. Step 5: Repeat the process until no more swaps are needed (array is sorted).

Time Complexity of Bubble Sort

The time complexity of Bubble Sort is as follows:

  • Best case: O(n) - When the array is already sorted.
  • Average case: O(n^2) - When the array is unsorted.
  • Worst case: O(n^2) - When the array is in reverse order.

Advantages of Bubble Sort

  • Simple and easy to understand.
  • In-place sorting algorithm (does not require additional memory).
  • Can detect whether the array is already sorted.

Disadvantages of Bubble Sort

  • Time complexity is O(n^2) in the average and worst cases, making it inefficient for large datasets.
  • Not suitable for large arrays due to poor performance.

Applications of Bubble Sort

  • Used in small datasets where the overhead of more complex algorithms (like Quick Sort or Merge Sort) is not worth it.
  • Helpful when the array is almost sorted or when only a few elements are out of place.
  • Used in educational settings to introduce students to the concept of sorting algorithms.
  • Can be applied in situations where the simplicity of the algorithm is more important than performance.

Bubble Sort Example

Consider the array: [5, 3, 8, 4, 2]

After each pass, the largest element "bubbles up" to its correct position:

  • Pass 1: [3, 5, 4, 2, 8] (8 bubbles to the end)
  • Pass 2: [3, 4, 2, 5, 8] (5 bubbles to its position)
  • Pass 3: [3, 2, 4, 5, 8] (4 is in its position)
  • Pass 4: [2, 3, 4, 5, 8] (Array is sorted)
